# Sanity

Integracion base del CMS y panel editorial del proyecto.

## Incluido

- `sanity.config.ts`
- `schemaTypes/documents`
- `schemaTypes/index.ts`
- ruta del Studio en `src/app/studio/[[...tool]]/page.tsx`
- cliente base en `src/lib/cms`

## Requisitos para dejarlo operativo

- completar `SANITY_PROJECT_ID` en `.env.local`
- confirmar `SANITY_DATASET`

## Studio

Ruta local esperada:

- `/studio`
